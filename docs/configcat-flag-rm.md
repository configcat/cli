# configcat flag rm
Delete flag
## Usage
```
configcat [options] flag rm
```
## Options
| Option | Description |
| ------ | ----------- |
| `--flag-id`, `-i`, `--setting-id`, `flag-id` | ID of the flag or setting to delete |
