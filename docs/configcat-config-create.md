# configcat config create
Create config
## Usage
```
configcat [options] config create
```
## Aliases
`cr`
## Options
| Option | Description |
| ------ | ----------- |
| `--product-id`, `-p` | ID of the product where the config must be created |
| `--name`, `-n` | Name of the new config |
