# configcat tag rm
Delete tag
## Usage
```
configcat [options] tag rm
```
## Options
| Option | Description |
| ------ | ----------- |
| `--tag-id`, `-i` | ID of the tag to delete |
| `--verbose`, `-v`, `/v` | Print detailed execution information |
| `-h`, `/h`, `--help`, `-?`, `/?` | Show help and usage information |

| Command | Description |
| ------ | ----------- |
