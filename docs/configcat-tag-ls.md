# configcat tag ls
List all tags
## Usage
```
configcat [options] tag ls
```
## Options
| Option | Description |
| ------ | ----------- |
| `--product-id`, `-p` | Show only a product's tags |
| `--verbose`, `-v`, `/v` | Print detailed execution information |
| `-h`, `/h`, `--help`, `-?`, `/?` | Show help and usage information |

| Command | Description |
| ------ | ----------- |
