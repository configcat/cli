# configcat config update
Update Config
## Usage
```
configcat [options] config update
```
## Aliases
`up`
## Options
| Option | Description |
| ------ | ----------- |
| `--config-id`, `-i` | ID of the config to update |
| `--name`, `-n` | The updated name |
