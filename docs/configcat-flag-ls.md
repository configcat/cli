# configcat flag ls
List all flags
## Usage
```
configcat [options] flag ls
```
## Options
| Option | Description |
| ------ | ----------- |
| `--config-id`, `-c` | Show only a config's flags |
| `--tag-name`, `-n` | Filter by a tag's name |
| `--tag-id`, `-t` | Filter by a tag's ID |
