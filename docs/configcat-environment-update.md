# configcat environment update
Update environment
## Usage
```
configcat [options] environment update
```
## Aliases
`up`
## Options
| Option | Description |
| ------ | ----------- |
| `--environment-id`, `-i` | ID of the environment to update |
| `--name`, `-n` | The updated name |
