# configcat product update
Update product
## Usage
```
configcat [options] product update
```
## Aliases
`up`
## Options
| Option | Description |
| ------ | ----------- |
| `--product-id`, `-i` | ID of the product to update |
| `--name`, `-n` | The updated name |

| Command | Description |
| ------ | ----------- |
