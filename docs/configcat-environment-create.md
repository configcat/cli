# configcat environment create
Create environment
## Usage
```
configcat [options] environment create
```
## Aliases
`cr`
## Options
| Option | Description |
| ------ | ----------- |
| `--product-id`, `-p` | ID of the product where the environment must be created |
| `--name`, `-n` | Name of the new environment |
