# configcat tag create
Create tag
## Usage
```
configcat [options] tag create
```
## Aliases
`cr`
## Options
| Option | Description |
| ------ | ----------- |
| `--product-id`, `-p` | ID of the product where the tag must be created |
| `--name`, `-n` | The name of the new tag |
| `--color`, `-c` | The color of the new tag |

| Command | Description |
| ------ | ----------- |
